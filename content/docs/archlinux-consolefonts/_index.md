## Introduction

ArchLinux’s [wiki page on console fonts](https://wiki.archlinux.org/index.php/Fonts#Console_fonts) was lacking an organized library of images available for previewing. So I made [this python script](consolefontsdump) to produce a PNG image of all fonts contained in `/usr/share/kbd/consolefonts`. Then I semi-manually built this page using vim and regexes.

Here is a tarball of the images&nbsp;: [fontspng.tar.gz](//static.alexandre.deverteuil.net/files/consolefonts/fontspng.tar.gz) (1&nbsp;359&nbsp;545 bytes). It expands to a “fontspng” directory containing the same 205 PNG files as below.

Please send any comments, suggestions or questions to [alexandre@deverteuil.net](mailto:alexandre@deverteuil.net).
